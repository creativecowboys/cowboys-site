## PAGE STRUCTURE (Same for all 3)

Every page follows this exact section order:

1. **Sticky Header**
2. **Hero Section** (headline + form LEFT, chat mockup RIGHT)
3. **Offer Banner** (full-width orange/gold bar)
4. **Pain Points** (3-column cards)
5. **How It Works** (4 steps LEFT + offer box RIGHT)
6. **Testimonials** (3-column cards)
7. **FAQ Accordion** (5 questions)
8. **Final CTA Section** (copy LEFT + form RIGHT)
9. **Footer**

## FORM SETUP NOTES

**Recommended: Formspree.io** (free tier, no backend needed)

1. Go to [formspree.io](https://formspree.io) → Create account
2. Create 3 forms: "Roofing Leads", "Plumbing Leads", "Law Firm Leads"
3. Each form gives you an endpoint like: `https://formspree.io/f/xyzabc12`
4. Set that as your form's action URL

**Required hidden fields on every form:**
- `industry` = "Roofing" / "Plumbing" / "Law Firm"
- `_subject` = "New Lead: [Industry]" (auto-sets email subject in Formspree)

**All forms need these fields:**
- Name (text, required)
- Business Name (text, required)
- Phone (tel, required)
- Email (email, required)
